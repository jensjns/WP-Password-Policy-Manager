Configure strong WordPress password policies for WordPress users to improve the security of WordPress

== Description ==

= Configure Password Age =
This plugin allows WordPress adminisrtators to configure the password age so WordPress users have to change their password once it expires. Such approach will improve the security of your WordPress because users are always changing the passwords.

= Password Policies for Stronger Passwords =
When a user's password expires, the user cannot:

* Use the same previous password
* Use the username as a password  

== Installation ==

1. Upload the `wp-password-policy-manager` folder to the `/wp-content/plugins/` directory
2. Activate the WP Password Policy Manager plugin from the 'Plugins' menu in the WordPress Administration Screens
3. Configure the Password Age from the Password Policy entry in the Settings menu

== Frequently Asked Questions ==

= How will a user know that his or her password is expired? =

If a WordPress user's password is expired, the user will be notified and asked to change the password upon logging in to WordPress.
